testing-calc
====

講義の最終課題のために実装した電卓アプリ

how to build
----

```
$ brew install node
$ npm install
$ npm run bower install
$ npm run gulp build
```

how to run
----

```
$ open public/index.html
```

how to test
----

```
$ brew install chromedriver
$ gem install bundler
$ bundle install
$ bundle exec rspec
```
